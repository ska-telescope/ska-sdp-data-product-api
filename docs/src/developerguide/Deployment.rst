Deployment Guide
~~~~~~~~~~~~~~~~

Kubernetes Deployment
=====================


The SDP Data Product API is deployed as part of the helm chart of the `SDP Data Product Dashboard <https://gitlab.com/ska-telescope/sdp/ska-sdp-dataproduct-dashboard>`_. Please see the deployment documentation in the `SDP Data Product Dashboard documentation <https://developer.skao.int/projects/ska-sdp-dataproduct-dashboard/en/latest/?badge=latest>`_ for more information.
