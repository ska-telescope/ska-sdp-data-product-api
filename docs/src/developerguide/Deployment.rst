Deployment Guide
================

Kubernetes Deployment
---------------------


The SKA Data Product API is deployed as part of the helm chart of the `SKA Data Product Dashboard <https://gitlab.com/ska-telescope/ska-dataproduct-dashboard>`_.
Please see the deployment documentation in the `SKA Data Product Dashboard documentation <https://developer.skao.int/projects/ska-dataproduct-dashboard/en/latest/?badge=latest>`_ for more information.
