Documentation for the SKA SDP Data Product API
=======================================================

The `ska-sdp-data-product-api` repository contains the code for the API that is used to access SDP Data Products in a storage volume.

Install the SKAO specific library
   > npm config set registry https://artefact.skao.int/repository/npm-internal/

   > yarn add @ska-telescope/ska-javascript-components@latest

   > npm config set registry https://registry.npmjs.org/
